Serviços providos pelo módulo RH.

  • Cadastrar novo servidor. URL

    • Passar os dados do novo servidor através do método POST.

    Exemplo:
    curl -u usuario:senha -H 'Content-Type:application/json' -X POST -d '{"nome": "nome", "user": "user", "email": "email"}' http://127.0.0.1:8000/api/servidor/

  • Listar todos os servidores. URL

    • Lista todos os servidores atualmente cadastrados.

    Exemplo:
    curl -u usuario:senha -H 'Content-Type:application/json' -X GET http://127.0.0.1:8000/api/servidores/json/

  • Pesquisar servidor a partir de matrícula. URL

    • Coloque a matrícula (id) do servidor que deseja pesquisar na URL.

    Exemplo:
    curl -u usuario:senha -H 'Content-Type:application/json' -X GET http://127.0.0.1:8000/api/servidor/1/json/

  • Alterar servidor. URL

    • Passar os dados do novo servidor através do método PUT.

    Exemplo:
    curl -u usuario:senha -H 'Content-Type:application/json' -X PUT -d '{"servidor_id": "1", "nome": "nome", "user": "user", "email": "email"}' http://127.0.0.1:8000/api/servidor/

  • Excluir servidor. URL

    • Coloque a matrícula (id) do servidor que deseja excluir na URL.

    Exemplo:
    curl -u usuario:senha -H 'Content-Type:application/json' -X DELETE http://127.0.0.1:8000/api/servidor/1/
